#Minecraft Server Starter Discord bot

## Setup:


1. download the GitHub source code
2. Configure the bot in the config.yml file
3. Run the bot.bat or the bo.sh file

if you can't run the script:
```
sudo chmod +x bot.sh
```

### Dependencies:

- discord.py
- mcstatus
- pyyaml

Install the dependencies:
```
pip install discord
pip install pyyaml
pip instal mcststus
